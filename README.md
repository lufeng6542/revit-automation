# Revit Automation - Claude Code Skill

通过 Revit MCP Server 控制 Autodesk Revit，18 种 MCP 操作覆盖模型查看、族放置、颜色标记、代码执行、视图导出、工作集同步。

## 安装

```bash
git clone https://github.com/lufeng6542/revit-automation.git ~/.claude/skills/revit-automation
```

需要: Revit 2018 + pyRevit + [Revit MCP Server](https://github.com/lufeng6542/revit-mcp-server)
